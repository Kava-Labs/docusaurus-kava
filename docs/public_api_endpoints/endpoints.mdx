# Kava Public Endpoints

Kava Labs provides public API endpoints for the community to use for consuming and integrating with Kava(both mainnet and testnet), e.g. for setting up and syncing a new archive node or getting the latest block info.

## Pruning API

Pruning APIs provide information on the last 1000 blocks, and are useful for getting the latest state and most recent transactions.

### Mainnet Pruning API Endpoints

- api.kava.io - HTTP Rest Endpoint
- rpc.kava.io - JSON-RPC Endpoint
- grpc.kava.io - GRPC Endpoint
- evm.kava.io - EVM JSON-RPC Endpoint
- wevm.kava.io - EVM Websocket Endpoint

### Testnet Pruning API Endpoints

- api.testnet.kava.io - HTTP Rest Endpoint
- rpc.testnet.kava.io - JSON-RPC Endpoint
- grpc.testnet.kava.io - GRPC Endpoint
- evm.testnet.kava.io - EVM JSON-RPC Endpoint
- wevm.testnet.kava.io - EVM Websocket Endpoint

## Archive API

Archive APIs provide a complete historical record of all state on the chain from the first block to the latest block.

### Mainnet Archive API Endpoints

- api.data.kava.io - HTTP Rest Endpoint
- rpc.data.kava.io - JSON-RPC Endpoint
- grpc.data.kava.io - GRPC Endpoint
- evm.data.kava.io - EVM JSON-RPC Endpoint
- wevm.data.kava.io - EVM Websocket Endpoint

### Testnet Archive API Endpoints

- api.data-testnet.kava.io - HTTP Rest Endpoint
- rpc.data-testnet.kava.io - JSON-RPC Endpoint
- grpc.data-testnet.kava.io - GRPC Endpoint
- evm.data-testnet.kava.io - EVM JSON-RPC Endpoint
- wevm.data-testnet.kava.io - EVM Websocket Endpoint
