# Default API Rate Limits

To provide fair use of public API capacity, request rate limits are enforced for consumers of the following endpoints

- [Mainnet Archive API](./endpoints)
- [Mainnet Pruning API](./endpoints)
- [Testnet Archive API](./endpoints)

## Mainnet Archive API Rate Limits

100 requests per 5 minutes.

## Mainnet Pruning API Rate Limits

1200 requests per 5 minutes.

## Testnet Archive API Rate Limits

500 requests per 5 minute period.
